# AI Course Lab 1
Course:Intro to AI - Semester3
Name: Ousmane Boubacar Bako

The purpose of this lab is to explore and compare different search algorithms for drone navigation on grid maps.

Part A: Focuses on uniform-cost grids where every move costs the same.

Part B: Extends to weighted grids with terrain costs (parks, highways, turbulence, construction zones).

The lab demonstrates how algorithms like BFS, UCS, Greedy Best-First, A\, and Weighted A\** behave in different environments, and highlights trade-offs between optimality, efficiency, and speed.
